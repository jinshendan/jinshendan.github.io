Data Processing

First of all, we have to load the MNIST dataset in Keras.

The dataset consists of training set and test set. The images are encoded as Numpy arrays, and the labels are an array of digits from 0 to 9. There is a one-to-one correspondence between images and labels.

Let’s have a look of the training data.

The dataset train_images is an array of 60000 images, each image is decribed by 28*28 pixels ranging from 0 to 255.

We can also visualize images in Python. Let’s display the first image in the dataset train_images.

The training data is formatted as float32 tensors of shape (60000, 2828), and the test data is formatted as float32 tensors of shape (10000, 2828)
